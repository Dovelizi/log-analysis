## designer BUG-01 Font Awesome 图标 tofu 方块修复 [P1]

- ⚠ 特异性坑: 我先写了 `.stat-card .text-Xxl { font-family: ... !important }` 想让大数字用衬线字，这条同时命中 `<i class="fas fa-database text-3xl">`（因为 class 同时含 "text-3xl"）→ 和 FA 兜底 !important 相同特异性冲突 → 必须加 `.stat-card i[class*="fa-"]` 提高特异性 + 把 FA 兜底规则放在后面双保险

- 📌 教训-CSS: 使用带 `!important` 的通配 class 选择器时（如 `[class*="text-"]`）必须意识到会命中任意 class 含此字符串的元素，会和其他 `!important` 规则因特异性相同而按源码顺序决胜
